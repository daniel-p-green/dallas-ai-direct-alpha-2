# Data Model

## Table: attendees

| Column | Type | Notes |
|---|---|---|
| id | uuid pk | generated by db |
| created_at | timestamptz | default now() |
| name | text | required |
| linkedin_url | text | optional, URL format |
| email | text | required, sensitive |
| ai_comfort_level | text | enum-like validation |
| help_needed | text[] | multi-select + Other |
| help_offered | text[] | multi-select + Other |
| honeypot | text | must remain empty |

## Public Projection
Use `attendees_public` view for directory reads. Exclude `email`, `honeypot`, and any future private notes.
